A lightweight RedM (VORP) script that allows admins to make themselves semi-invisible via command.


__**DE**__

 :ghost: Become a Ghost (Admin Only) 

Ein leichtgewichtiges RedM (VORP) Script, das es **Admins** ermöglicht, sich per Command **halb unsichtbar** zu machen.

 :lock: Zugriff

* Nur für Gruppe: **admin**
* Andere Spieler: keine Nutzung, komplett silent

 :gear: Funktionen

* :heart: Unverwundbar als Geist
* :ghost: Halb-Unsichtbarkeit
* :bricks: Kollision bleibt aktiv
* :arrows_counterclockwise: StateBag Sync (OneSync)
* :no_bell: Keine Chat- oder UI-Ausgaben

:video_game: Nutzung

```
/ghost
```

 :link: Logging

* Discord Webhook integriert
* :green_circle: Aktiviert / :red_circle: Deaktiviert
* Spieler + Charakter + Zeit

 :zap: Performance

* Eventbasiert
* Kein Loop
* Sehr ressourcenschonend

---

:ghost: Minimalistisch, performant und perfekt für Admins


__**EN**__

A lightweight RedM (VORP) script that allows admins to make themselves semi-invisible via command.

:lock: Access

* For use only by the **admin** group

* Other players: no use, completely silent

 :gear: Features

 * :heart: Invulnerable as a spirit
* :ghost: Semi-invisibility
* :bricks: Collision remains active
* :arrows_counterclockwise: StateBag Sync (OneSync)
* :no_bell: No chat or UI output

:video_game: Usage

```
/ghost
```

 :link: Logging

* Discord Webhook integrated
* :green_circle: Enabled / :red_circle: Disabled
* Player + Character + Time

*  :zap: Performance

* Event based
* No loop
* Very resource-saving

:ghost: Minimalist, high-performance, and perfect for admins
